# Networking Fundamentals with Device Configuration & Home Network Security Labs

## Overview

This repository contains a collection of hands-on networking labs completed using **Cisco Packet Tracer**. The labs focus on networking fundamentals, device configuration, network communication, routing, remote management, DHCP, FTP services, web connectivity, NAT, and home network security.

These practical exercises strengthened my understanding of enterprise networking concepts and improved my ability to configure, manage, troubleshoot, and secure network environments.

---

# Objectives

- Understand networking fundamentals.
- Configure LAN and home network environments.
- Configure routers and switches.
- Implement DHCP and NAT.
- Configure FTP and Web Server communication.
- Perform IP and MAC address analysis.
- Configure Telnet and SSH for remote management.
- Troubleshoot network connectivity.
- Analyze traffic flow and packet routing.
- Strengthen home network security concepts.
