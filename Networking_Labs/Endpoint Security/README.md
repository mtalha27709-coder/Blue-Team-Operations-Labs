# Cisco Packet Tracer – Endpoint Security Labs

## Overview

This repository contains the Packet Tracer labs completed as part of the **Cisco Networking Academy – Endpoint Security** course. These hands-on labs were designed to strengthen practical skills in securing endpoints, protecting wireless networks, implementing access control, ensuring data integrity, and troubleshooting network connectivity issues using Cisco Packet Tracer.

The labs focus on real-world security concepts that are commonly applied in enterprise environments to improve network protection and endpoint security.

---

# Objectives

- Understand endpoint security fundamentals.
- Configure and verify wireless security using WPA2.
- Implement Access Control Lists (ACLs) to restrict unauthorized access.
- Explore secure file transfer and data encryption concepts.
- Verify file integrity using hash-based techniques.
- Understand Packet Tracer's Logical and Physical workspace.
- Diagnose and resolve common wireless connectivity issues.

---

# Labs Completed

## 1. ACL – Access Control Demonstration

### Description
Configured and tested Standard and Extended Access Control Lists (ACLs) to control network traffic based on security requirements.

### Tasks Performed
- Configured Standard ACLs.
- Configured Extended ACLs.
- Applied ACLs to router interfaces.
- Allowed and denied traffic based on IP addresses.
- Verified ACL functionality using ping and connectivity tests.

### Skills Learned
- Access Control Lists (ACL)
- Network Traffic Filtering
- Router Security
- Security Policy Implementation

---

## 2. Basic Wireless Security (WPA2)

### Description
Configured a secure wireless network using WPA2-PSK authentication and connected wireless clients securely.

### Tasks Performed
- Configured Wireless Router.
- Set SSID.
- Enabled WPA2 Personal Security.
- Configured Strong Passphrase.
- Connected wireless devices.
- Verified successful secure communication.

### Skills Learned
- WPA2 Security
- Wireless Authentication
- Secure Wi-Fi Configuration
- Wireless Client Connectivity

---

## 3. File Data Encryption (FTP)

Explored secure file transfer concepts using FTP services while understanding the importance of protecting sensitive data through encryption techniques.

### Tasks Performed
- Configured FTP Server.
- Connected FTP Client.
- Uploaded files.
- Downloaded files.
- Verified successful file transfer.
- Studied encryption concepts for protecting transmitted data.

### Skills Learned

## 4. File Integrity Checks

### Description
Performed integrity verification to ensure files remained unchanged during storage or transmission by comparing cryptographic hash values.

### Tasks Performed
- Generated file hashes.
- Compared original and modified files.
- Verified integrity status.
- Observed how hash values change after file modification.

### Skills Learned
- File Integrity Verification

- Data Validation
- Cybersecurity Best Practices
