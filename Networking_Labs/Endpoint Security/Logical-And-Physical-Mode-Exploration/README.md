# Packet Tracer - Logical and Physical Mode Exploration



## 📌 Overview


This project demonstrates the exploration of both **Logical Mode** and **Physical Mode** in Cisco Packet Tracer. The lab helps build a strong understanding of how network devices are connected logically and how they are arranged physically in real-world environments.


The project focuses on understanding network topology visualization, device placement, cable connections, and physical infrastructure simulation inside Packet Tracer.



---



## 🎯 Objectives



- Explore Logical Mode in Cisco Packet Tracer
- Understand Physical Mode and device placement
- Visualize network topology and connections
- Learn how network devices communicate logically
- Simulate real-world physical infrastructure
- Understand cable management and device organization
