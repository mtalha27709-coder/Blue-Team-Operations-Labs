# Network Defense Security Labs

## Overview

This repository contains a complete collection of hands-on Network Defense and Security labs implemented using Cisco Packet Tracer. The project focuses on securing enterprise networks through access control, firewall technologies, traffic monitoring, wireless hardening, IoT physical security, and centralized logging techniques.

These labs were completed as part of practical cybersecurity and network defense training to strengthen real-world networking and security implementation skills.

* Configure Access Control

* Configure Named Standard IPv4 ACLs

* Configure Wireless Router Hardening and Security

* Logging from Multiple Sources
