# Cisco IOS Navigation Lab (Packet Tracer)



## 🧪 Overview


This project is a hands-on Cisco Packet Tracer lab focused on learning how to navigate Cisco IOS. It helps build foundational networking skills required for SOC analysts and network engineers.



---



## 🎯 Objectives


- Establish console connection between PC and switch
- Access Cisco IOS CLI
- Explore User EXEC and Privileged EXEC modes
- Use context-sensitive help (`?`)
- Practice command completion (Tab key)
- Configure system clock using IOS commands



---



## 🛠 Tools Used

- Cisco Packet Tracer
- Cisco IOS CLI (Switch S1)
- Console cable connection



---



## 📚 Skills Learned


- CLI navigation in Cisco IOS
- EXEC mode differences (User vs Privileged)
- Help system usage in IOS
- Command syntax understanding
- Basic system configuration (clock setting)


---


## 🔥 Key Commands Practiced
?
enable
configure terminal
clock set
show clock


---

## 📌 What You Learn from This Lab


- How Cisco devices are managed using CLI
- How to troubleshoot using IOS help system
- How to switch between modes safely
- How real network engineers operate switches



---



## 🚀 SOC/NOC Relevance


This lab builds foundation for:
- Network monitoring
- Log analysis
- Device configuration understanding
- SOC Tier-1 networking skills
