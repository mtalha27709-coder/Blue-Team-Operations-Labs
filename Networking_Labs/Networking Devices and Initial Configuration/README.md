# Cisco Packet Tracer ñ Networking Devices and Initial Configuration Labs

## Overview

This repository contains the Packet Tracer labs completed as part of the **Cisco Networking Academy ñ Networking Devices and Initial Configuration** course. These hands-on labs focus on configuring Cisco networking devices, understanding core networking protocols, verifying IPv4 and IPv6 connectivity, performing remote device management using SSH, and analyzing communication between hosts.

The labs provide practical experience in configuring routers and switches while developing a strong foundation in enterprise networking and network administration.

---

# Objectives

- Understand the role of networking devices.
- Perform initial router and switch configuration.
- Navigate the Cisco IOS Command Line Interface (CLI).
- Configure secure remote management using SSH.
- Analyze ARP communication.
- Verify dual-stack (IPv4 & IPv6) connectivity.
- Understand TCP and UDP communication.

# Labs Completed

## 1. ARP Analysis Lab

### Description
Analyzed the Address Resolution Protocol (ARP) to understand how devices resolve IP addresses into MAC addresses before communication within a local network.

### Tasks Performed
- Generated network traffic using ping.
- Observed ARP Request and ARP Reply messages.
- Examined the ARP table.
- Verified MAC address resolution.
- Understood Layer 2 and Layer 3 communication.

### Skills Learned
- ARP Protocol
- MAC Address Resolution
- LAN Communication
- Packet Analysis
- Network Troubleshooting
