# 🚀 Packet Tracer - Initial Router Configuration Lab



## 📌 Overview


This project demonstrates the fundamental configuration of a Cisco router using Packet Tracer. It covers essential tasks required to bring a router from its default state to a secure and operational device.


---



## 🎯 Objectives


- Verify default router configuration
- Configure initial router settings
- Secure CLI access with passwords
- Set up MOTD (Message of the Day) banner
- Save running configuration to NVRAM
- Backup configuration to flash memory



---



## 🛠️ Lab Tasks



### 🔹 Part 1: Verify Default Configuration


- Established console connection to router (R1)
- Accessed privileged EXEC mode
- Analyzed running configuration
- Checked startup configuration in NVRAM

- 

### 🔹 Part 2: Configure Router Settings


- Set hostname to **R1**
- Configured MOTD banner:
- Unauthorized access is strictly prohibited.

- - Secured access:
- Enable password (encrypted)
- Console password
- Enabled password encryption
- Verified configuration using CLI commands



### 🔹 Part 3: Save Configuration


- Saved running configuration to NVRAM: copy running-config startup-config
- - Verified saved configuration
- (Optional) Backed up configuration to flash memory
